## 标准流程

```mermaid
flowchart LR
    A[创建文档分支] --> B[提交 PR]
    B --> C[内容评审]
    C --> D[CI 检查]
    D --> E[合并 main]
    E --> F[Mintlify 自动部署]
    F --> G[用户可见]
```

## 分支策略

- `main`：生产文档
- `docs/*`：文档功能分支
- 所有变更通过 PR 合并

## 评审清单

- 是否与产品当前行为一致
- 是否提供最小可执行示例
- 是否包含风险与限制说明
