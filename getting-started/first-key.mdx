## 步骤 1：选择 Key 类型

支持 LLM、Code PAT、SaaS、SSH、Cloud、Payment 等类型。  
如果不确定，先选 LLM 或直接粘贴 Key。

## 步骤 2：粘贴并自动识别

将 Key 粘贴到输入框后，应用会尝试自动识别供应商（例如 `sk-` -> OpenAI）。

## 步骤 3：保存到 Keychain

点击保存后，Key 值进入 macOS Keychain，界面仅展示必要元数据。

## 步骤 4：一键复制使用

在菜单栏或主窗口复制 Key，用于终端、IDE 或 Agent。

<Tip>建议开启剪贴板自动清除，降低误泄漏风险。</Tip>
