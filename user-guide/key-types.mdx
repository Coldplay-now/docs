## Key Types

常用类型包括：

- LLM
- Code PAT
- SaaS
- SSH
- Email SMTP
- Cloud
- Payment
- Generic

## 何时用 Generic

当供应商不在预置列表，或你想自定义字段时，使用 Generic 最灵活。

## 命名建议

- 名称格式：`<project>-<provider>-<env>`
- 示例：`console-openai-prod`
- 避免使用无语义名称（如 `key1`、`new-token`）
